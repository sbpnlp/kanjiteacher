\subsection{Hiragana \cjk{ひらがな}}
\label{sec:hiragana}

Hiragana is a syllabic script. A full table of the Hiragana alphabet:

\begin{CJK}
  \begin{tabular}{c c c c c c c}
 &a&i&u&e&o&n\\
$\emptyset$&あ&い&う&え&お&ん\\
k&か&き&く&け&こ&\\
s&さ&し&す&せ&そ&\\
t&た&ち&つ&て&と&\\
n&な&に&ぬ&ね&の&\\
h&は&ひ&ふ&へ&ほ&\\
m&ま&み&む&ね&も&\\
y&や& &ゆ& &よ&\\
r&ら&り&る&れ&ろ&\\
w&わ& & & &を&\\
  \end{tabular}
\end{CJK}
Adapted from \shortcite{Hadamitzky1995}.
